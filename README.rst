==================================================
``nbmerge``: merge / concatenate Jupyter notebooks
==================================================

Installation
============

.. code:: sh

    pip install nbmerge

Usage
=====


.. code:: sh

    nbmerge file_1.ipynb file_2.ipynb file_3.ipynb > merged.ipynb
